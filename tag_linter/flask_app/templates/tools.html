Tag Relations

These tools define information about tags, and relationships between tags. They are saved in a database, so when you restart the server, they will be kept around. Use these for long-term planning.

  • Junk Tags — Tags that provide no info and should be removed.
  • (WIP) Soft Parents — Like tag parents, but less strict.
  • (WIP) Tag Disambiguation — Take a vague tag, and define alternatives to replace it with
  • (WIP) Helicopter Parents — Tags that must always be with one of their children. For example, dress must always be with red dress, green dress, etc...

Others

These tools are single operations, useful for simple, one-time operations

  • Search & Destroy — Remove a tag or a namespace from a tag service.
  • Transpose Namespace — Replace all foo:abcd tags with bar:abcd tags.
